# 背景音乐目录

将您的背景音乐文件（.mp3 格式）放在此目录中。

## 支持的格式

- MP3 (.mp3) - 推荐

## 推荐的免费背景音乐来源

- **Pixabay Music**: https://pixabay.com/music/ - 免费商用，无需署名
- **Free Music Archive**: https://freemusicarchive.org/ - 多种音乐风格
- **Incompetech**: https://incompetech.com/music/royalty-free/ - Kevin MacLeod 的免费音乐
- **YouTube Audio Library**: https://www.youtube.com/audiolibrary - YouTube 官方免费音乐库
- **Bensound**: https://www.bensound.com/ - 免费使用需署名

## 推荐的音乐风格

对于教育视频，建议使用：

- **轻音乐 / Lo-Fi** - 轻松、不干扰注意力
- **古典钢琴曲** - 优雅、适合学习氛围
- **环境音乐 (Ambient)** - 低调的背景音
- **轻快电子乐** - 适合轻松愉快的内容

## 注意事项

1. 确保您有使用这些音乐文件的合法权利
2. 建议使用时长超过 3 分钟的音乐文件（自动循环）
3. 背景音乐的音量建议设置为 10%-30%，不要盖过语音
4. 文件名不支持中文和特殊字符（建议使用英文/数字）

## 示例

将您的 mp3 文件放在这里后，系统会自动从中随机选择一首作为背景音乐。
